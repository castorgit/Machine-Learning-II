# Machine Learning II Course Syllabus

## Course Information
**Course Title:** Machine Learning II - Advanced Topics  
**Semester:** Spring 2026  
**Credits:** 3  

## Course Description
This course covers advanced machine learning concepts building upon foundational ML knowledge. Students will explore deep learning, unsupervised learning, reinforcement learning, and modern ML architectures.

## Prerequisites
- Machine Learning I or equivalent
- Strong programming skills in Python
- Linear algebra, calculus, and probability/statistics
- Basic understanding of neural networks

## Learning Objectives
By the end of this course, students will be able to:
1. Implement and optimize deep neural network architectures
2. Apply unsupervised learning techniques to real-world problems
3. Design and train convolutional and recurrent neural networks
4. Understand and implement reinforcement learning algorithms
5. Evaluate and improve model performance using advanced techniques
6. Deploy machine learning models in production environments

## Course Schedule

### Week 1-2: Advanced Supervised Learning
- Ensemble methods (Bagging, Boosting, Stacking)
- Gradient boosting (XGBoost, LightGBM, CatBoost)
- Model optimization and hyperparameter tuning

### Week 3-4: Unsupervised Learning
- Dimensionality reduction (PCA, t-SNE, UMAP)
- Clustering algorithms (K-means, DBSCAN, Hierarchical)
- Anomaly detection

### Week 5-7: Deep Learning Fundamentals
- Neural network architectures
- Backpropagation and optimization algorithms
- Regularization techniques (Dropout, Batch Normalization)
- Transfer learning

### Week 8-10: Convolutional Neural Networks
- CNN architectures (ResNet, VGG, EfficientNet)
- Computer vision applications
- Object detection and segmentation

### Week 11-12: Recurrent Neural Networks
- RNN, LSTM, GRU architectures
- Sequence modeling
- Natural language processing applications
- Attention mechanisms and Transformers

### Week 13-14: Advanced Topics
- Generative models (GANs, VAEs)
- Reinforcement learning basics
- Model deployment and MLOps
- Ethical AI and bias mitigation

### Week 15: Final Project Presentations

## Grading Breakdown
