# Labs

This directory contains hands-on lab exercises for practical learning.

## About Labs
Labs are weekly exercises designed to:
- Reinforce lecture concepts
- Provide hands-on coding practice
- Explore ML libraries and tools
- Prepare you for assignments
- Build your ML portfolio

## Lab Structure
Each lab includes:
- Jupyter notebook with exercises
- Dataset or data loading instructions
- Step-by-step guidance
- Practice problems with hints
- Extension challenges (optional)

## How to Use Labs

### Setup
1. Navigate to the lab directory
2. Open the Jupyter notebook
3. Install required libraries (see notebook)
4. Follow instructions in the notebook

### Completion
- Complete all TODO sections in the notebook
- Run all cells to verify your solutions
- Answer discussion questions
- Try extension challenges for extra practice

### Submission
- Labs contribute to your participation grade
- Save completed notebooks with your name
- Commit to your forked repository
- No formal submission required, but track your progress

## Getting the Most from Labs
✅ **Do the labs yourself** - copying defeats the purpose  
✅ **Experiment** - try different approaches and parameters  
✅ **Debug actively** - understand errors, don't just fix them  
✅ **Ask questions** - labs are for learning, not grades  
✅ **Take notes** - document insights for future reference  
✅ **Help others** - explaining concepts reinforces learning

## Troubleshooting
**Kernel crashes?** Reduce data size or batch size  
**Import errors?** Check requirements.txt and reinstall  
**Unexpected results?** Review lecture materials and notebook hints  
**Stuck?** Ask for help - that's what we're here for!
