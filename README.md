## Tech Stack
*   **Language:** Python 3.x
*   **Libraries:** `scikit-learn`, `XGBoost`, `LightGBM`, `CatBoost`, `Pandas`, `NumPy`, `Matplotlib`, `Seaborn`.
*   **Environment:** Jupyter Notebooks / Google Colab.

## How to Use This Repo
1. **Lectures:** Navigate to the specific session folder (e.g., `/05-GBM`) to find the Jupyter Notebooks
2. **Datasets:** Most datasets can be found in the `/data` folder or via the provided Kaggle links within the notebooks.
3. **Homework:** Instructions for assignments are in Blackboard
